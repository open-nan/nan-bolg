# vue 渐进式javasrcipt框架

## 什么是渐进式
渐进式是指我们开始使用该框架的一部分功能来满足我们的需求，在后期需求增加时我们可以对框架的其他功能进行拓展，我们不用一开始就全面的采用该框架的所有功能和特性，这使得我们的整个系统变的更灵活，像这样逐步集成的方式就叫做渐进式
